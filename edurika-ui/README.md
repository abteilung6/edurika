## Getting Started

### Install

Install dependencies.

```bash
npm install
```

Serve with hot reload

```bash
npm run dev
```

### Generate

Generate openapi client

```bash
npm run generate-openapi-client
```

### Lint

```bash
npm run lint
```

### Typecheck

```bash
npm run typecheck
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```

View and interact with your tests via UI.

```bash
npm run test:ui
```
